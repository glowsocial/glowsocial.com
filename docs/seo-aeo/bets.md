# Glow Social SEO/AEO Bet Log

This log tracks SEO/AEO bets, not just completed tasks.

Daily jobs should add one entry per shipped pass. Keep entries short. The weekly review uses this file plus Search Console/analytics/repo evidence to decide whether `learnings.md` should change.

## Entry template

```md
## YYYY-MM-DD — Page or cluster name

- **URL:** https://glowsocial.com/...
- **Work type:** existing-page refresh | calculator/tool support | comparison page | AEO question | internal links | indexing/canonical | content launch
- **Bet:** What buyer/search behavior are we testing?
- **Why this page:** Why this page was the best next target.
- **Change made:** Short factual summary.
- **Expected signal:** What should improve and where to look later: impressions, clicks, CTR, rankings, preview CTA clicks, assisted signup, internal-link flow, indexing.
- **Evaluation window:** Usually 2–6 weeks depending on indexing.
- **Do not overlearn if:** What uncertainty could explain a weak result.
- **Changed URLs for indexing:**
  - https://glowsocial.com/...
```
